# Assignment 1 — Finance, AI, and Risk Detection

This repository documents my process of using an AI writing assistant (Claude Code) to draft and refine a short piece about the intersection of finance and artificial intelligence, specifically around risk and fraud detection. It includes the raw AI-generated draft, my own edited final version, and a reflection on how the collaboration between me and the assistant evolved across both files.

## Repository Contents

- `assistant_draft.md` — Content generated with the help of Claude Code, including a reflection note on the prompting process
- `final_version.md` — My own revised version of the draft, with personal edits, added specifics, and a note explaining what I changed and why
- `README.md` — This file
